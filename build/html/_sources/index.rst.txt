Welcome to Sunstone's documentation!
====================================

.. toctree::
   :maxdepth: 2
   :caption: Contents:

Sunstone is a python3 module designed to aid in Müller polarimetry with focus on analysis of biological samples. The base module ``sunstone`` provides the classes ``Jones`` (Jone's vectors), ``Stokes`` (Stokes vectors) and ``Muller`` (Müller matrices).
